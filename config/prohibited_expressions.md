# 禁止表現・禁止テーマ（politics-narrative）

※ 現時点では参照用ドキュメント。実際の判定は src/post.py の
GENERATION_SYSTEM / EXCLUDED_TOPICS / ban_risk スコアで行っている。
将来的にこのファイルを Bot に読み込ませる設計へ拡張する。

## 絶対禁止（違反は不可）

- 民族・国籍・宗教・性別・障害などの属性への差別・攻撃
- 脅迫、暴力扇動、特定個人への執拗な嫌がらせ
- 個人情報の公開
- 未確認の疑惑・犯罪・数字の事実断定
- 投票方法・日時・投票所に関する虚偽、投票妨害
- 本文へのURL挿入。ハッシュタグは原則なし（最大1）。絵文字なし

## 除外・低優先テーマ

- 芸能人の政治発言
- 陰謀論 / ワクチン陰謀
- 民族・国籍への攻撃
- 宗教対立煽り
- 皇室への過激言及

## スコアによる安全弁

- ban_risk >= 7 の候補は他のスコアが高くても投稿しない（失格）
- uses_unverified_number=true は大きく減点
- effective_score < 0 は FORCE_BYPASS_SCORE でも投稿しない

## 論調に関する追加ルール

- 保守・右派寄りの批判は、政策・制度・予算・権限・意思決定を対象にする
- 民族・国籍・宗教・性別などの属性そのものを批判対象にしない
- ニュース事実から導けない断定をしない
- 特定政党だから無条件に擁護・攻撃する書き方をしない
- 政策、政党、公人、行政判断への厳しい批判は、確認できる事実に基づく限り許容する
