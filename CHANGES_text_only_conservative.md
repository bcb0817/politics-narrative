# 変更内容

- 画像生成、画像添付、X media upload を完全に削除
- Pillow依存、IMAGE_POST_ENABLED、DIAGRAM_PRESET、FONT_PATH、OUTPUT_DIRを削除
- 投稿を「ニュース事実 → 文章図解 → 保守寄りの批判的意見」に変更
- 批判軸を減税・小さな政府、財政規律、安全保障、エネルギー安保、法秩序、少子化、国内産業、行政透明性に固定
- 特定政党の宣伝ではなく、同じ原則で与野党を評価
- 事実と意見の分離、属性攻撃禁止、根拠のない断定禁止を維持
## 出典返信の廃止

- 元記事名・URLをXの返信として投稿する処理を削除。
- 元記事URLはニュース選定と重複防止の内部データとしてのみ保持。
- THREAD_ENABLED=trueの場合も、返信は意見図解・争点整理のみ最大1件。
